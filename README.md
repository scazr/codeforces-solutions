# Codeforces Solutions

A collection of accepted solutions to [Codeforces](https://codeforces.com/) problems, written in Python and C++.

## Structure

```
accepted/
└── {problem_id}.py / .cpp
```

Files are named directly by their Codeforces problem ID (e.g. `2094C.cpp` → [Problem 2094C](https://codeforces.com/problemset/problem/2094/C)).
